Arduino Functions and Library Functions Overview

Category Function Definition / Purpose Syntax Example Code
User Defined Functions General Function Structure Custom functions created by the user to organize code and perform repeated tasks. returnType functionName(parameters) {
statements;
}		 int add(int a, int b) {
return a + b;
}
Void Function Does not return a value; used to perform an action. void functionName() { ... } void blinkLED() {
digitalWrite(13, HIGH);
delay(500);
digitalWrite(13, LOW);
delay(500);
}
Function With Parameters Allows passing values to functions for flexible control. void func(int x, int y) void blinkCustom(int pin, int d) {
digitalWrite(pin, HIGH);
delay(d);
digitalWrite(pin, LOW);
delay(d);
}
I/O Functions pinMode() Configures a pin as INPUT, OUTPUT, or INPUT_PULLUP. pinMode(pin, mode); pinMode(13, OUTPUT);
digitalWrite() Writes HIGH or LOW to a digital pin. digitalWrite(pin, value); digitalWrite(13, HIGH);
digitalRead() Reads HIGH or LOW from a digital input pin. digitalRead(pin); int x = digitalRead(2);
analogRead() Reads analog voltage from A0–A5 and returns 0–1023. analogRead(pin); int value = analogRead(A0);
analogWrite() Writes PWM output (0–255) to pins 3,5,6,9,10,11. analogWrite(pin, value); analogWrite(9, 128); // 50% brightness
analogReference() Selects voltage reference for analog input. analogReference(type); analogReference(INTERNAL);
Character Functions isAlpha() Returns true if character is alphabetic (A–Z, a–z). isAlpha(c); if (isAlpha('A')) Serial.println("Alpha");
isAlphaNumeric() Returns true if character is letter or digit. isAlphaNumeric(c); isAlphaNumeric('7'); // true
isDigit() Checks if c is numerical digit (0–9). isDigit(c); isDigit('3'); // true
isHexadecimalDigit() Checks if character is valid hex (0–9, A–F). isHexadecimalDigit(c); isHexadecimalDigit('F');
isSpace() Returns true if character is a space. isSpace(c); isSpace(' ');
isWhitespace() Returns true for whitespace (space, tab, newline). isWhitespace(c); isWhitespace('\n');
isUpperCase() Checks if character is uppercase letter. isUpperCase(c); isUpperCase('A');
isLowerCase() Checks if character is lowercase letter. isLowerCase(c); isLowerCase('b');
Math Functions abs() Returns absolute value of number. abs(x); int y = abs(-10);
constrain() Limits a value within a minimum and maximum range. constrain(x, min, max); value = constrain(value, 0, 255);
max() Returns the larger of two values. max(a, b); int m = max(3, 9);
min() Returns the smaller of two values. min(a, b); int m = min(3, 9);
pow() Raises a number to a power. pow(base, exponent); float p = pow(2, 3);
sqrt() Returns square root. sqrt(x); float r = sqrt(49);
